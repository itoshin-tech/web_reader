DeepL APIキー取得ガイド

プレミアムプランをご購入いただいた方は、Google翻訳よりも自然で高精度な「DeepL」翻訳エンジンを使用できます。これにはDeepL公式サイトから発行される「APIキー」の設定が必要です。
💡 知っておいてほしいこと
DeepLには無料（API Free）プランがあり、月間50万文字までは完全無料で利用可能です。クレジットカードの登録が必要な場合がありますが、枠を超えない限り料金は発生しません。

取得・設定の流れ

1
DeepL API の登録ページへ移動 DeepL公式サイトのAPI登録ページへアクセスします。
DeepL API 公式サイトへ
2
「無料で登録」を選択してアカウント作成 「DeepL API Free」プランを選択し、メールアドレスと支払い情報（本人確認用）を入力してアカウントを作成してください。
3
APIキーを表示・コピー アカウント管理画面の「アカウント」タブの下部にある「DeepL API用認証キー」という項目を探します。
例: 12345678-abcd-...:fx
4
アプリの設定画面に入力 WebVoiceアプリを開き、[設定] > [翻訳 engine] > [キー入力] 欄にコピーしたキーを貼り付けて保存してください。
✓ 設定完了！
これで、記事を読み上げる際に最高品質の翻訳を楽しめるようになります。

よくある質問

Q. キーを入力してもエラーになります。
A. キーの末尾が :fx で終わっているか確認してください（Freeプランの場合）。また、前後に余計なスペースが含まれていないかチェックしてください。

Q. 料金がかかることはありますか？
A. Freeプランであれば、月間50万文字を超えて翻訳できないよう制限されているため、勝手に課金されることはありません。安心してご利用いただけます。
